## 🧩 Sobre o projeto

Este site foi desenvolvido com o objetivo de **informar, acolher e facilitar o acesso a conteúdos sobre o Transtorno do Espectro Autista (TEA)**.

A plataforma foi pensada para diferentes públicos:
- Responsáveis por crianças
- Pessoas autistas
- Usuários com hipersensibilidade sensorial
- Pessoas com dificuldades de leitura
- Cidadãos em busca de direitos garantidos por lei

💡 O site busca oferecer uma experiência acessível, com conteúdo claro, leitura fluida e informações confiáveis.

---

## Objetivo

Promover **informação, inclusão e acessibilidade digital**, contribuindo para uma sociedade mais empática e consciente.
